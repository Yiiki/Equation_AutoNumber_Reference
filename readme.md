# 自动编号公式方法

*导出时间: 2025/10/8 00:38:42*

---

### User

word里如何给公式自动连续编号

---

### Anser  

🧷 step1 : 插入自动编号域：


按下 Ctrl + F9，会出现一对花括号 { }；


在括号中输入：
nginx复制编辑SEQ eq \\* ARABIC



然后选中 { SEQ eq \\* ARABIC }，按 F9 更新域。


编号就会自动变为 (1)。


将编号复制粘贴到公式输入框的井号后面，回车即可。


若插入或删除公式，只需全选文档并按 F9 更新全部编号。



🧷 step2：用“插入”实现公式引用


选中编号 → “插入 → 书签” → 命名为 Eq1、Eq2 等。

🧷 step3：在正文中引用：


光标放在想插入引用的位置；


“引用 → 交叉引用”；


引用类型选择“编号项”；


插入你想引用的书签（例如 Eq2）。
