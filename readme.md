在 Microsoft Word 中给公式自动连续编号有几种方法，以下是最常用的几种做法，适合中英文论文排版：

🧮 方法一：利用“公式 + 右对齐制表位 + 域代码”实现自动编号（推荐）
这是最标准、通用且自动编号的方式。
步骤：


在公式前按下 Enter 开一新行。


插入公式：


用 Alt + = 打开公式编辑器（或点击“插入 → 公式”）。




在公式右侧对齐编号：


选中该行，点击菜单栏 “开始 → 段落 → 制表位”；


在“制表位位置”中输入一个合适的值（如 15cm），选择“右对齐”，点击“确定”。


按下 Tab 键，光标就会跳到右边。




插入自动编号域：


按下 Ctrl + F9，会出现一对花括号 { }；


在括号中输入：
nginx复制编辑SEQ eq \* ARABIC



然后选中 { SEQ eq \* ARABIC }，按 F9 更新域。


编号就会自动变为 (1)。




后续每插入一个新的公式并重复第 4 步，编号会自动递增：(2)、(3)…。


若插入或删除公式，只需全选文档并按 F9 更新全部编号。



🧷 方法二：用“插入→题注”实现公式编号（适合论文）


选中公式；


选择菜单栏 “引用 → 插入题注”；


在“标签”处新建一个“公式”标签；


在“位置”选择“所选项下方”；


Word 会自动生成如 “公式 (1)” 的编号；


若添加或删除公式，Word 可自动重新编号（通过“更新域”）。



✅ 优点：与图表编号统一管理，可在“交叉引用”中引用公式编号。
❌ 缺点：格式稍显复杂，不如域代码灵活。


⚙️ 方法三：使用表格排版（适合固定格式）


插入一个两列的单行表格；


左列放公式，右列放编号；


右列输入 (1)；


用前述 SEQ eq 域让右列自动编号。



适用于要求公式和编号严格对齐的期刊模板。


💡 附加技巧


若要让编号带章节号（如“(2.3)”），可用：
nginx复制编辑SEQ eq \* ARABIC \s 1

并确保章节号用多级列表自动编号。


更新所有编号：
Ctrl + A 全选 → F9 更新域。
